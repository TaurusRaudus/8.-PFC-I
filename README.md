# Recommendations for taking the subject
- Select papers within the state of the art (generally shouldnt be more than 3 years old)
- Search for models which share the same objective (It is expected them solve the same problem but in a different way)
- Select Models which provide code to enhance the reproducibility
- Select Models which use the same datasets to enhance the reproducibility
